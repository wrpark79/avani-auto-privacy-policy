# Avani Auto 개인정보 처리방침

**최종 수정일: 2026-04-24**

Avani Auto(이하 "서비스")는 이용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다.
본 개인정보 처리방침은 서비스 이용 과정에서 개인정보가 어떻게 처리되는지 안내합니다.

## 1. 수집하는 개인정보 항목

서비스는 이용자가 직접 입력하는 아래 정보를 처리합니다.

- **차량 정보**: 차량명, 차량 번호판, 연식
- **정비 기록**: 정비 부품명, 교체 주기, 주행거리, 정비 날짜, 비용, 메모

차량 번호판은 개인을 식별할 수 있는 정보에 해당할 수 있으며, 이에 따라 개인정보로 취급합니다.

## 2. 개인정보 처리 방식

이용자가 입력하는 모든 정보는 **이용자의 기기 내 로컬 저장소에만 저장**되며, 외부 서버로 전송되지 않습니다.
서비스 운영자는 이용자의 데이터를 열람하거나 수집할 수 없습니다.

## 3. 제3자 제공 및 외부 공유

서비스는 이용자의 데이터를 제3자에게 제공하거나 공유하지 않습니다.

- 광고 플랫폼을 연동하지 않습니다.
- 분석 플랫폼(GA 등)을 사용하지 않습니다.
- 외부 클라우드에 저장하지 않습니다.

## 4. 데이터 보관 및 삭제

이용자가 입력한 데이터는 기기 내에만 저장됩니다.
앱을 삭제하거나 앱 내 설정에서 해당 데이터를 삭제하면 모든 정보가 기기에서 제거됩니다.
서비스 운영자는 이용자의 데이터를 별도로 보관하지 않습니다.

## 5. 개인정보 보호를 위한 조치

서비스는 이용자 데이터 보호를 위해 다음을 적용합니다.

- 데이터는 기기 내부 저장소에만 저장됩니다.
- 외부 네트워크로 전송하는 기능은 구현되어 있지 않습니다.
- 운영자가 접근할 수 없는 구조로 유지됩니다.

## 6. 이용자의 권리

이용자는 언제든지 앱 내 데이터를 직접 삭제하거나 앱을 삭제하여 저장된 모든 정보를 제거할 수 있습니다.
